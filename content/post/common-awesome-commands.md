* Mode4 + Enter, new terminal
* Mode4 + r, run a program
* Mode4 + C, close the program
* Mode4 + j or k, next or prev window
* Mode4 + left or right, next or prev tab
* Mode4 + number, go to the tab
* Mode4 + m, manifier to full screen
