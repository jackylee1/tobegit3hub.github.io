* bin 普通用户，开机会用到
* boot mlinux核心档案
* grub 开机设定
* dev 设备文档
* etc 软件设置
* initd 软件启动脚本
* home 各用户
* lib 各种目标档与函数库
* modules 不同版本核心的模组
* media 暂存
* mnt 挂在区
* opt 第三方软件
* proc 内存中的core和程序
* root 超级管理员的
* sbin 开机或维护模式能用的系统
* src 网络服务提供的资料
* sys 内存中的core和档案系统
* usr 包括X11R6, bin, include, lib, local, sbin, share, src, man, doc
* var 包括cache, lib, lock, log, spool, run
* lost+found 不正常关机
